REQUIREMENTS:
  - Python 3.7 or higher installed
  - Python added to system PATH
  - Administrator access (optional, for package installation)

INSTALLATION STEPS:
  1. Navigate to the project directory:
     cd "c:\Users\Yulark\OneDrive\Desktop\Group"
     
  2. Install dependencies:
     pip install -r requirements.txt
     
  3. Run the application:
     - Option A: Double-click run.bat
     - Option B: Run command: python run_server.py
     
  4. Open browser to: http://localhost:5000
  
  5. Login with demo credentials (see above)
  
  6. Explore the dashboard and test scenarios

STOPPING THE SERVER:
  - Press CTRL+C in the command prompt/terminal
