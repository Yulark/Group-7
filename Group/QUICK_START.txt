API ENDPOINTS:
================================================================================

Authentication:
  POST   /api/login                 - User login
  POST   /api/logout                - User logout
  GET    /api/user-info             - Get current user information

Access Control:
  POST   /api/request-access        - Request access to resource
  GET    /api/test-scenarios        - Get available test scenarios
  POST   /api/run-test/<id>         - Run specific test scenario
  GET    /api/access-summary        - Get access statistics

Admin:
  GET    /api/users                 - List demo users

All endpoints return JSON responses with appropriate HTTP status codes.
